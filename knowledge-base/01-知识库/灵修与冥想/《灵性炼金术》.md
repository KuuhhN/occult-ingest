# 《灵性炼金术》— 潘蜜拉·克里柏

## 📌 一句话概述

## 🔑 核心观点

1. **光之工作者唯一的任务是疗愈自己** ——当你成为清明的管道，光自然透过你流向世界
2. **情绪是孩子，感受是老师** ——情绪不应批判或压抑，应以意识之光拥抱
3. **灵性炼金术 = 体验情绪 + 中立观察** ——这就是疗愈的本质
4. **一开始就没有罪** ——耶稣之死不是赎罪，是无知和恐惧导致的人间悲剧
5. **真正的治疗与技巧无关** ——关键是透过内在成长获得「解决问题的频率」
