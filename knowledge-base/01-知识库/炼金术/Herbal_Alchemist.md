# 《The Herbal Alchemist's Handbook》— Karen Harrison

## 📌 一句话概述

> 融草药学、占星术、卡巴拉和炼金术于一体的实战手册——从行星时间计算到配方调配，全套可操作的草药魔法体系。

## 📖 全书结构

### 基础理论（1–5 章）
- 四大元素 / 七行星 / 行星日与时 / 草药行星对应 / 卡巴拉十源质

### 实践配方（6–19 章）
- 熏香、仪式油、护符、药浴、魔药、香囊、香粉等全套操作流程

### 附录 A–H
- 对应表、配方库、物料清单（全书最实用部分）

## 🔑 核心观点

1. 草药同时具有物理疗效和能量效用，祖先们从不在两者之间划界
2. 正确的行星日+行星时操作 = 能量双倍放大
3. 卡巴拉十源质可作为配方设计框架
4. 炼金术的核心——solve et coagula（分离与重新结合）——同样适用于草药处理
