# 埃及神庙每日供奉仪式 — 导读

## 📌 一句话总结

UCL 皮特里博物馆藏的柏林 3055 号纸莎草译本——记录卡纳克阿蒙神庙的每日供奉仪式流程，由 HonoYoku（秘义之光）中译。是现存最完整的地中海世界每日神庙仪式文本之一。

## 🎯 核心要点

1. **柏林 3055 号纸莎草**：记录了底比斯（卡纳克）阿蒙-拉神庙的每日供奉公式——"由首要的纯洁祭司在他的服务日进行"
2. **仪式九步骤**：从净化准备→开启神龛→供奉食物/饮料/衣物/油膏→清洁→封闭神龛→扫除脚印
3. **仪式目的**："维护宇宙的经纬和进程"（维持 Maat）——古埃及人不认为自己在"崇拜偶像"，而是通过圣像与神圣存有互动
4. **图文并茂**：皮特里博物馆的砂岩碑、木碑等实物照片穿插其中，展示了 Mehy 举手唱诵拉神赞歌、法老用香油涂抹阿蒙神像等场景
5. **PGM 的直接祖先**：这种神庙供奉仪式是后来希腊魔法纸莎草中大量供奉和召唤仪式的远古模板
