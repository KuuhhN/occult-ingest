# Cat Magick: Bast — 导读

## 📌 一句话总结

Anousen Leonte 的 Bast（埃及猫女神）符印魔法小册子——通过现代 sigil magic 技术召唤 Bast 的灵性精华，19 页的精简短篇。

## 🎯 核心要点

- **Bast/Bastet**：埃及猫女神，太阳女神，保护者和丰饶之神
- **Sigil Magick**：通过绘制符印（sigil）并"充能/遗忘"来激活潜意识的操作
- **作者另有 Gnostic Magick 等著作**
