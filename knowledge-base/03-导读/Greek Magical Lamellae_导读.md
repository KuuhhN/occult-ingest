# Greek Magical Lamellae — 导读

## 📌 一句话总结

Roy Kotansky 编订的希腊魔法金属符片（金/银/铜/青铜薄片）学术全集第一卷——收录所有已知出处的刻字金属护符，含希腊原文、英译和考古学评注。是 PGM 纸莎草文献之外最重要的希腊魔法原始材料合集。

## 🎯 核心要点

1. **Lamellae = 魔法金属薄片**：与 PGM 纸莎草不同，这些是刻在贵金属上的符咒文本——通常折叠成小片，穿绳佩戴或置入墓葬
2. **"已知出处"的第一卷**：本卷仅收录考古出土地明确的样本，确保了学术可靠性
3. **从护身符到咒符**：收录的 Lamellae 涵盖驱邪、爱情咒、竞技咒、医疗护符等多种类型
4. **跨文化桥梁**：许多 Lamellae 显示埃及、希腊、犹太元素的融合——"三倍伟大的赫尔墨斯"等 syncretic 神名频繁出现
5. **Papyrologica Coloniensia 系列**：德国北威州科学院出版的顶级古典学丛书

## 🗺️ 知识背景

- **所属体系**：魔法实践 / 古典学 / 考古学
- **难度等级**：★★★★★（学术文献，需古希腊语基础）
- **作者**：Roy Kotansky（独立学者，希腊魔法铭文研究的权威）
- **丛书**：Papyrologica Coloniensia Vol. 22/1
- **语言**：古希腊语 + 英语
- **页数**：467 页
