# 《Splendor Solis》— 导读

> 📕 [[Splendor_Solis.pdf|📖 点击打开原版 PDF]]（英文扫描版）| 中文导读

## 📌 一句话总结

> 《Splendor Solis》（太阳的光辉）是 16 世纪最著名的炼金术手抄本，本书首次以全彩印刷呈现其权威英译本，并附三位学者的专题论文，从历史、作者、帕拉塞尔苏斯运动、图像评注四个维度全面解读这部炼金术杰作。

---

## 🎯 核心要点

1. **Splendor Solis 是什么** — 16 世纪德国炼金术插图手抄本，含 22 幅精美寓言彩图，描绘「伟大工作」（Magnum Opus）的完整过程
2. **黄金黎明的复兴** — 20 世纪初，赫尔墨斯黄金黎明结社将其重新发掘，马瑟斯曾出版注释版
3. **作者之谜** — 传统归名于所罗门·特里斯莫辛（Salomon Trismosin），但最新研究对此提出质疑
4. **与帕拉塞尔苏斯的关联** — Splendor Solis 在帕拉塞尔苏斯运动中被视为权威文本，对医学炼金术影响深远
5. **图像与文本的结构** — 手抄本并非杂乱无章的箴言集，而是有精心设计的文本结构和图像学体系
6. **四阶段重复呈现** — 「伟大工作」的黑化→白化→黄化→红化在手抄本中以多种方式反复呈现
7. **全彩版的价值** — 此前市面版本多为 1920 年的黑白复制版，本书是英国图书馆藏 Harley MS 3469 的首次完整全彩呈现

- **所属体系：** 炼金术 / 赫尔墨斯主义 / 帕拉塞尔苏斯医学

- **难度等级：** 🟡 进阶
- **适合人群：** 对炼金术图像学、手抄本研究、文艺复兴神秘学感兴趣的读者

## 💡 关键术语表

| 术语 | 含义 |
|------|------|
| **Splendor Solis** | 拉丁语「太阳的光辉」，16 世纪炼金术插图手抄本的书名 |
| **Harley MS 3469** | 大英图书馆藏 Splendor Solis 最权威手稿版本 |
| **Salomon Trismosin** | 传统上被认定为 Splendor Solis 的传奇作者，德国炼金术士 |
| **Paracelsus** | 帕拉塞尔苏斯（1493–1541），瑞士医生、炼金术士，将炼金术转向医学应用的关键人物 |
| **Golden Dawn** | 黄金黎明赫尔墨斯结社，1888 年成立的魔法结社，对 20 世纪神秘学复兴影响巨大 |
| **Magnum Opus** | 拉丁语「伟大的工作」，炼金术核心操作流程的总称 |
