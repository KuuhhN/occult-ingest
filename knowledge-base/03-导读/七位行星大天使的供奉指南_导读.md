# 七位行星大天使的供奉指南 — 导读

## 📌 一句话总结

西方实践魔法师（匿名）的博客系列——由 HonoYoku（穹印之源/秘义之光）中译，记录了个人供奉七位行星大天使（Iophiel/木星等）的实际仪式流程、祷文、供品制作和灵性体验。

## 🎯 核心要点

1. **实践者的第一手记录**：不是教科书或魔典——是一个真实魔法师在家中进行行星天使工作的博客日志，包含照片描述、食谱、具体时间和位置
2. **行星天使体系**：聚焦七位传统行星大天使（木星 Iophiel → 火星 Samael → 太阳 Michael 等）
3. **供品制作指南**：木星蛋糕的完整食谱（四种谷物+牛奶，400°F 烘烤 10 分钟）、供盘的摆放方向（蓝色盘→木星对应）
4. **El 祷文**：以古代迦南至高神 El 的英文祷文开场——这是一种兼容多神论的"最高源头"框架
5. **保护性魔法实操**：萨麦尔（火星大天使）用于建立房产保护屏障的具体步骤
