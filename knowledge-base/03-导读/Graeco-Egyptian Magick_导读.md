# Graeco-Egyptian Magick — 导读

## 📌 一句话总结

Tony Mierzwicki 的希腊-埃及日常魔法手册——以 PGM 和希腊化时期的仪式为蓝本，构建了"每日赋权"（Everyday Empowerment）的灵性实践体系。序言由《Modern Magick》作者 Donald Michael Kraig 撰写。

## 🎯 核心要点

1. **每日赋权**：全书定位在"日常可执行的灵性实践"——区别于一次性的大仪式，强调每天小规模的仪式和冥想
2. **希腊-埃及混合魔法体系**：与 Hermetic Magic 和 Seven Spheres 共享 PGM 的 syncretic 框架，但更偏向日常生活应用
3. **Kraig 序言**：Donald Michael Kraig（《Modern Magick》——20 世纪末最畅销的仪式魔法入门教材）作序，将本书定位在西方仪式魔法的现代普及传统中
4. **实践驱动**：含具体的祈祷文、冥想脚本、神圣名字列表、每日行星时对应表
