# 《The Herbal Alchemist's Handbook》— 导读

> 📕 [[Herbal_Alchemist.pdf|📖 点击打开原版 PDF]]（英文）| 页码 1–188 | 中文导读

## 📌 一句话总结

> 一部将草药学、占星术、卡巴拉和炼金术融为一体的实战手册——不是理论书，是从原料到成品、从行星时间到仪式配伍的完整操作指南。

---

## 🎯 核心要点

1. **草药炼金术 = 草药 + 行星能量 + 仪式意图** — 每株植物对应特定行星和元素，正确的采收时间（行星日/行星时）和配方比配能放大其能量
2. **Solve et Coagula** — 「分离与重新结合」，炼金术的核心操作也是草药处理的哲学：提取精油、混合配方、注入意图
3. **完整的操作体系** — 从行星日/时的计算，到精油、熏香、药浴、护符的配方设计，到手写仪式的步骤排列，全部覆盖
4. **卡巴拉生命之树的十源质** — 作为配方设计的框架，每源质对应特定行星和草药
5. **19 章正文 + 8 个附录** — 附录才是真宝藏：行星对应表、元素对应表、完整配方库、所需物料清单
6. **不仅是药，也是魔法** — 草药同时具有物理疗效和能量效用，祖先们从不在两者之间划界

---

## 🗺️ 知识背景

- **所属体系：** 炼金术 / 草药魔法 / 占星术 / 卡巴拉
- **前置知识：** 需要基本的占星符号和行星概念；中文读者可先读《炼金术》（徐德伟）入门
- **难度等级：** 🟡 进阶——操作性强，但需要交叉运用多个体系
- **适合人群：** 想亲自动手实践草药炼金术/自然魔法的读者

## 💡 关键术语表

| 术语 | 含义 |
|------|------|
| **Spagyric Art** | 炼金术在 16 世纪的别称，来自拉丁语 solve et coagula（分离与重新结合） |
| **Planetary Day / Hour** | 行星日/行星时——每天每小时由一颗行星掌管，是配方时机的核心依据 |
| **Qabalah** | 卡巴拉生命之树体系，本书用作配方的十源质框架 |
| **Philtre** | 魔药/爱情药水——带有特定意图的液体配方 |
| **Tincture** | 酊剂——以酒精萃取草药活性成分的液体 |
| **Elixir** | 灵药——结合了药用和仪式意图的液体配方 |
