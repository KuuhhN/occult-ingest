# 《英灵魔法》— 导读

> 📕 [[英灵魔法.pdf|📖 点击打开原版 PDF]] | 页码 1–103

## 📌 一句话总结

> 来自 The Gallery of Magick 的混沌魔法实战手册——教你创造、喂养和运用「英灵」（servitor，意念构装体），让它们替你在现实世界中完成特定任务。

---

## 🎯 核心要点

1. **英灵是什么** — 由你创造的意念实体，拥有名字、外观、性格和特定任务，像「心灵机器人」一样为你工作
2. **不需要天赋** — 这套方法刻意设计为初学者也能上手，核心要素是清晰的意图、开放的心态和遵循指定步骤
3. **越具体越好** — 英灵的目标必须明确具体，模糊的目标（如「变有钱」）产生模糊的结果
4. **英灵有生命周期** — 创造→喂养→工作→完成任务→消散。不给它设定终结时间，它可能徘徊不去
5. **团队协作** — 可以创建一队英灵各自负责不同领域，也可以创建临时英灵完成一次性任务
6. **属于混沌魔法传统** — 不拘泥于任一宗教/神秘学体系，实用主义至上，什么有效用什么

## 💡 关键术语表

| 术语 | 含义 |
|------|------|
| **英灵**（Servitor） | 由魔法师用意念创造的实体，拥有特定任务和有限的生命周期 |
| **混沌魔法**（Chaos Magic） | 不以任一宗教或体系为核心的魔法流派，强调个人经验和实用效果 |
| **意志/意图**（Will/Intent） | 英灵的核心驱动力——你给它什么指令，它就执行什么 |
| **喂养**（Feeding） | 通过注意力、能量或仪式维持英灵的存在和力量 |
| **消散**（Dismissal） | 任务完成后主动解散英灵，释放其能量 |
